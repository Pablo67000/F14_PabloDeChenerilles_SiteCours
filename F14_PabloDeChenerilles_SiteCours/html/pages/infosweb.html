Le web a été crée en 1991 par Time Berners-Lee avec la venue du HTML. Le CSS est venu compléter le HTML en 1996.

Une page qui s'affiche dans votre navigateur est définie par le code écrit en HTML et CSS

HTML (HyperText Markup Language) = Sert à gérer et organiser le CONTENU de la page web
CSS (Cascading Style Sheets) = Sert à gérer l'APPARENCE de la page web
Le CSS repose sur le HTML, on commence donc par le HTML

Les outils de programmation répandus sont les suivants: Notepad, Notepad++, SublimText

WYSIWYG (What You See Is What You Get):
Ce sont des programmes qui se veulent faciles d'emploi, ils permettent de créer des sites web sans apprendre de langage particulier.
Exemples: Mozilla Composer, Microsoft Expression Web etc.

Navigateurs: Quand on code un site web, on se doit d'utiliser plusieurs navigateurs pour que notre site web fonctionne pour une majorité d'utilisateurs, au minimum Google Chrome et Mozilla Firefox qui sont les deux navigateurs les plus répandus.
