Balises
HTML langage de balise, repéré par < et >.

Balises paires
S'ouvrent, contiennent texte, et se ferment plus loin:
<titre>Ceci est un titre</titre>

Balises orphelines
Servent le plus souvent à insérer un élément à un endroit précis (ex:image). Pas nécessaire de délimiter le début et la fin de l'image
on veut juste dire à l'ordinateur "insère une image ici"
s'écrit:
<image/> ou <image> mais on recommande le slash pour bien différencier des balises paires.

Attributs

Attributs = options des balises. Viennent compléter pour donner des infos suppl. Se place après nom de balise ouvrante et a le plus souvent une valeur,
comme ceci:
<balise attribut="valeur">
Ex: <image nom="photo.jpg"/>, nom de l'image à afficher.

Dans le cas de balise paire, on ne met les attributs QUE dans balise ouvrante, pas dans la fermante.

Créer des liens

Un site web est composé de plusieurs pages.
Un lien est un texte sur lequel on peut cliquer pour se rendre sur une autre page.
On peut faire un lien d'une page a.html vers une page b.html, on peut aussi faire un lien vers un autre site. Balise <a> ; attribut href ; valeur de l'attribut est URL de la page ciblée.

Lien vers un autre site
Ex : <a href= « https://google.com »>Google</a>
Par défaut s'affiche en bleu souligné. Si page déjà ouverte, violet.

Lien dans même dossier, ou dossier plus haut.
Dossier plus haut : <a href="../troisiemepage.html">Page 3</a>

Un lien vers une ancre

Une ancre est une sorte de point de repère que l'on peut mettre dans nos pages HTML lorsqu'elles sont très longues. Il peut alors être utile de faire un lien amenant plus bas dans la même page.
Pour créer une ancre il faut rajouter l'attribut id à une balise qui va alors servir de repère.
Ce peut être n'importe quelle balise, un titre par exemple. On l'utilise pour donner un nom à l'ancre.
Cela nous servira ensuite pour faire un lien vers cette ancre.
Par exemple : <h2 id=« mon_ancre »>Titre</h2>

Ensuite il suffit de créer un lien, mais l'attribut href contiendra un dièse # suivi du nom de l'ancre.
Ex : <a href=«#mon_ancre »>Aller vers l'ancre</a>

Normalement en cliquant ça nous amènera plus bas dans la même page.

Lien vers une ancre située dans une autre page
Faire un lien qui ouvre une autre page ET qui amène directement à une ancre située plus bas sur cette page. Saisir le nom de la page, suivi d'un dièse (#), suivi du nom de l'ancre.
Par exemple : <a href= «ancres.html#rollers »>

Un lien qui affiche une infobulle au survol
Ajouter l'attribut facultatif title

Un lien qui ouvre une nouvelle fenêtre
Ajouter target = «_blank »
Selon la configuration du navigateur, la page s'affichera dans une nouvelle fenêtre ou un nouvel onglet. Vous ne pouvez pas choisir entre l'ouverture d'une nouvelle fenêtre ou d'un nouvel onglet. NB : Le visiteur lui-même peut décider s'il veut ouvrir le lien dans une nouvelle fenêtre.
Il fera Maj + Clic sur le lien pour ouvrir dans une nouvelle fenêtre ou Ctrl+Clic pour ouvrir dans un nouvel onglet. Un lien pour envoyer un e-mail
<a href= «mailto:lemail_cible »>Envoyez un e-mail!</a>

Un lien pour télécharger un fichier
<a href= « monfichier.zip »>Télécharger le fichier</a>

Insérer image :
Dépends du format de l'image, sur internet toutes les images sont compressés.
Format JPG ou JPEG, peut supporter jusqu'à 16 millions de couleurs.
Format PNG (Portable Network Graphic) plus récent et plus performant des formats, l'utiliser des que possible.
Avantage : l'image peut être transparente et on n'altère pas sa qualité. Existe en 2 versions : 8 bit, 256 couleurs, et 24 bits 16 millions de couleurs.
La compression du JPEG étant plus optimisé pour les photos, privilégier ce format.
Format GIF, image animée.

Insérer une image :
Balise impaire <img> attribut src= «nomimage.jpg(ou png gif)»
attribut alt=

La miniature cliquable :
Plus attrayant, plus d'indications sur la page.

exemple final : <a href="C:\Users\Pablo\Documents\web\TP\html\naruto.jpg" target="_blank" title=AgrandirNaruto><img src="C:\Users\Pablo\Documents\web\TP\html\narutoresize.jpg" alt="NarutoMiniature"/></a>
